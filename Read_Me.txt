This is the repo conataining all the work and files that were made in preparing the semantically similar shloka recommendation system.


This folder contains 3 sub-folders.
The Subfolders are:
Scrapping: Having Codes for the first step of Data Collection where Data from Various sites was collected.
Making of Database : This includes arrangement of data in meaningful format so that ideal recommendation could be extracted from the raw data collected from web-scrapping.
ML Part: This part has codes of all the ML Models and approaches applied on the test data and the results of them.
